Polymarket 预测市场数据工具

一个用于获取和分析Polymarket预测市场数据的现代化Web应用

数据获取

实时获取Polymarket平台上的预测市场数据

智能筛选

支持按分类、状态、价格等多维度筛选市场

数据导出

支持JSON和CSV格式的数据导出功能

主要功能
  • 实时市场数据获取和显示
  • 响应式Web界面，支持移动设备
  • 市场分类和筛选功能
  • 分页浏览和搜索功能
  • JSON/CSV数据导出
  • 市场统计和分析
  • RESTful API接口
  • 配置管理和系统监控
技术栈
后端技术
  • Flask - Python Web框架
  • Flask-CORS - 跨域资源共享
  • Flask-RESTful - REST API扩展
  • py-clob-client - Polymarket API客户端
前端技术
  • HTML5/CSS3/JavaScript - Web基础技术
  • Bootstrap 5 - UI框架
  • Bootstrap Icons - 图标库
  • Axios - HTTP客户端
API接口
端点 方法 描述
/api/v1/markets GET 获取市场列表（支持分页和筛选）
/api/v1/markets/{id} GET 获取单个市场详情
/api/v1/markets/categories GET 获取市场分类列表
/api/v1/markets/stats GET 获取市场统计信息
/api/v1/config GET 获取当前配置
/api/v1/config PUT 更新配置
/api/v1/health GET 健康检查
使用指南
快速开始
  1. 访问主页查看市场列表
  2. 使用筛选器找到感兴趣的市场
  3. 点击市场标题查看详细信息
  4. 使用导出功能下载数据
筛选技巧
  • 使用搜索功能快速定位特定市场
  • 按分类筛选（政治、体育、加密货币等）
  • 勾选"仅显示活跃市场"查看当前可交易市场
  • 调整显示数量控制页面加载速度
